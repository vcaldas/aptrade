Aptrade
=======

Another Python library for backtesting built on top of Backtrader.

This project starts as a fork of the Backtrader library, but it has evolved
into a separate project with its own features and improvements. The main goal
of Aptrade is to provide a more user-friendly and efficient backtesting
experience for traders and developers and hopefully to keep compatibility with
the original Backtrader library.
